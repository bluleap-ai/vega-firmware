# VEGA PROJECT - AIR QUALITY

## Run example for ZMOD4510, BME688, AS7331 sensors:

```
cargo run --bin vega-zmod
```
```
cargo run --bin vega-bme
```
```
cargo run --bin vega-as7331
```
